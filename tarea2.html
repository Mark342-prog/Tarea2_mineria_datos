Varianza explicada

variance_explained <- svd_result$d^2 / sum(svd_result$d^2)

Reduccin de dimensionalidad (SVD truncado)

Seleccionamos k componentes principales.

k <- 20

U_k <- svd_result$u[,1:k]
D_k <- diag(svd_result$d[1:k])
V_k <- svd_result$v[,1:k]

approx_matrix <- U_k %*% D_k %*% t(V_k)

dim(approx_matrix)
## [1]  943 1682

Error de reconstruccin

reconstruction_error <- norm(rating_matrix - approx_matrix, type = "F")

## [1] 812.8734

Un menor error indica mejor aproximacin usando k componentes.

Interpretacin de resultados

La grfica de los valores singulares muestra que los primeros componentes tienen valores mucho mayores que el resto. Esto indica que una gran parte de la informacin del sistema de calificaciones est concentrada en pocas dimensiones. En otras palabras, aunque el dataset tiene muchas pelculas, las preferencias de los usuarios pueden resumirse en un nmero menor de factores importantes.

En la grfica de varianza acumulada se observa que los primeros componentes explican un alto porcentaje de la variabilidad total. Esto significa que no es necesario usar todas las dimensiones originales para representar adecuadamente la informacin, ya que una parte significativa puede capturarse con menos componentes.

Al reconstruir la matriz utilizando solo 20 componentes, se obtiene un error de reconstruccin moderado. Esto indica que existe cierta prdida de informacin, pero la estructura general del sistema de ratings se mantiene. Esto demuestra que el comportamiento de los usuarios puede representarse mediante factores latentes que resumen patrones de preferencia.

La visualizacin obtenida mediante t-SNE muestra una clara tendencia a la formacin de dos grupos principales, correspondientes a los diagnsticos benigno (B) y maligno (M). Esto indica que las variables del dataset contienen informacin suficiente para diferenciar ambos tipos de tumores.

Se observa que los puntos malignos tienden a concentrarse en una regin especfica del espacio bidimensional, mientras que los benignos ocupan otra zona distinta. Aunque existe cierto solapamiento entre ambos grupos, la separacin general es evidente.

Dado que t-SNE preserva relaciones locales entre observaciones, esta agrupacin sugiere que los pacientes con caractersticas similares tienden a compartir el mismo diagnstico. Esto demuestra la utilidad del algoritmo como herramienta de visualizacin para explorar la estructura interna de datos de alta dimensionalidad.

Interpretacin de resultados

La proyeccin obtenida mediante UMAP muestra una separacin clara entre los casos benignos y malignos. A diferencia de t-SNE, UMAP tiende a preservar no solo las relaciones locales entre puntos, sino tambin parte de la estructura global del conjunto de datos.

En la visualizacin se observa que los dos grupos presentan menor solapamiento y una estructura ms compacta, lo que sugiere que UMAP captura de forma eficiente la geometra interna de los datos.

Esto confirma que las caractersticas clnicas medidas en el dataset permiten diferenciar ambos diagnsticos y demuestra que UMAP es una herramienta efectiva para la reduccin de dimensionalidad y visualizacin de datos complejos.

Interpretacin de resultados

Las seales ECG originales corresponden a registros reales del MIT-BIH Arrhythmia Database. Estas seales representan mediciones elctricas cardacas registradas mediante mltiples derivaciones, las cuales pueden contener mezclas de diferentes fuentes fisiolgicas y posibles artefactos.

Al aplicar ICA, el algoritmo busca descomponer las seales observadas en componentes estadsticamente independientes. En la visualizacin de los componentes recuperados se observan patrones diferenciados, lo que indica que ICA logr separar parcialmente las fuentes subyacentes presentes en la seal original.

Este resultado demuestra la utilidad de ICA en el anlisis de seales biomdicas, ya que permite identificar estructuras ocultas y separar posibles fuentes independientes a partir de mezclas lineales observadas.
